ENSF 409 Project

This project is for the course ENSF 409 - Principles of Software Development. This project simulates a course registration system for a school. It uses Java as the main system and MySQL for the simulated database system. The program prompts the user to view, search, and register into courses of their choice. Little bit of MySQL is used to store the different courses that are offered in the school. The source files can be found here: https://github.com/EddieSKim/ENSF_409.git

ENSF 480 Project

This project is for the course ENSF 480 - Principles of Software Design. This project simulates a movie theatre seat reservation system. It uses Java as the main system and uses a text based database. The program allows user to pick a certain movie, and choose the available showtime. The user then picks an available seat and confirms the checkout for payment. The source files can be found here: https://github.com/EddieSKim/ENSF480-Ticket-Reservation_Project.git
